# Live Bot 直播弹幕机器人 — Figma 提示词

每段都是 **完整自包含** 的提示词。把第 §F0 共享设定追加到每条 F1–F9 末尾一起粘进 Figma "Make Designs"。所有界面文案为简体中文。

设计要点：
- 桌面端基准 1080×700，最小 920×580
- 视觉语言：液态玻璃 + 漂浮高斯色团 + 半透明白卡
- 导航：单段 7 项，无右侧面板
- 自动化用 6 张 AutoCard 网格
- 主题：浅色 / 深色 两态
- 主色预设 4 个
- 自定义颜色选择器：HSL 三滑块 + HEX（无色环，节省纵向空间）

## §F0 · 共享设定（追加到每条提示词末尾）

```
界面语言：简体中文，所有标签、按钮、提示、占位符必须为中文。
基准画布：1080 × 700 桌面端。
字体：PingFang SC / SF Pro（中文正文）+ SF Mono（房号、UID、时间戳、JSON）。
基础字号 12px，标题 17/700，小字 10/600 letter-spacing 0.4。

视觉语言：macOS Big Sur 之后的 Liquid Glass —— 半透明白卡 + 漂浮高斯色团 + 镜面高光。

【背景】整窗 160deg 线性渐变：
- 浅色：from #eaeef5 to #dfe3eb
- 深色：from #14141b to #0c0c10
背景上漂浮 3 个 460–520px 高斯色团（radial-gradient + Gaussian Blur 80），所有 Frame 共享：
- 左上 主色团（浅色透明度 70% / 深色 78%）位置 x=-160 y=-180
- 右上 #ff2d55 玫红团（浅 82% / 深 85%）位置 右上角向外 -120
- 右下 #34c759 翠绿团（浅 83% / 深 86%）位置 右下角向外
建议在 Figma 中先建一个 BackgroundBlobs 组件，每个 Screen Frame 第一层放一份实例。

【布局】2 区，无右侧面板：
- 顶栏（高 52）：玻璃带，linear-gradient #ffffff80→#ffffff50（浅色）/ #ffffff0e→#ffffff04（深色）
- 左侧栏（宽 196，可折叠）：玻璃 #ffffff7a→#ffffff48（浅）/ #ffffff10→#ffffff05（深），右 1px 细描边
- 主区：可滚动，padding 14

【7 项导航单段】仪表盘 / 登录与房间 / 监听与发送 / 自动回复 / AI 机器人 / 数据统计 / PK 与活动 / 系统与更新
NavItem 高 44，圆角 12：默认透明 / 激活主色半透明渐变（透明度 0.84-0.92）+ 主色描边（透明度 0.7）+ 左侧 28×28 主色实底图标块白图标。

【主题】仅 浅色 / 深色 两态，命名后缀 " · 浅色"" · 深色" 并排。
【主色预设 4 个】iOS 蓝 #007aff（默认）/ 翠绿 #34c759 / 橙 #ff9500 / 玫红 #ff2d55。

【液态玻璃组件规范】
- Glass 卡片：fill 半透明白渐变（浅 #ffffffd8→#ffffffaa / 深 #ffffff14→#ffffff09），圆角 18，1px 描边（浅 #00000014 / 深 #ffffff1c），阴影 0 6px 22px #0000001c，顶部 1px 镜面高光线（浅 #ffffffe6 / 深 #ffffff30）从 x:6 到 x:parent-12。
- Btn 胶囊高 30 圆角 15：
  · 主按钮：linear-gradient(180deg, accent.brighter 10%, accent.darker 4%) + accent.darker 18% 描边 + 10px accent 透明阴影
  · 默认按钮：linear-gradient #ffffffd6→#ffffff8a + 1px rim + 4px 阴影
  · 顶部 1px 镜面高光（x:5, w:parent-10, h:1）
- IconBtn 圆形 30×30：半透明白渐变，激活时主色半透明背景。
- Toggle 36×20 胶囊：开态 linear-gradient accent.brighter 8%→accent + 8px 主色阴影；关态 linear-gradient #d4d4d9.darker→#d4d4d9。圆形拇指 16×16 白渐变 + 4px 黑阴影。

【字号档】
- 标题 17/700 letter-spacing 0
- 区段标题 11/700 letter-spacing 0.6
- 正文 12/500
- 副文 10/600 letter-spacing 0.4
- 等宽 SF Mono 11/500

【关键约束】
- 卡片永远半透明，能透出底色团 — 不要把卡填实色
- 不用密集表格，全部用卡片网格 + chip 列表
- 没有右侧面板：实时状态、日志、用户查询都收进主区卡片
- 间距舒展，padding 12-14，元素呼吸感
```

## 使用建议

1. **先在 Foundations Page 里建一个 BackgroundBlobs 组件**：3 个 460-520 大圆 + radial-gradient + Gaussian Blur 80。后续每个 Screen Frame 第一层都嵌入它，不然玻璃感全失。
2. 把 §F0 共享设定 **追加到每条 F1–F9 末尾** 一起粘进 Figma "Make Designs"。
3. 顺序：F1 仪表盘（验证玻璃感）→ F9 主题 popup（确定色彩）→ F2..F8。
4. 如果 Make Designs 把卡片画成实色 dashboard 风格，立刻用追加指令"将所有卡片改为半透明白渐变填充，让背景色团透出来"修正。
5. Slint 版只做浅色 / 深色两态，不需要"跟随系统"画面。
